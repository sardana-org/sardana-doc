Macro Logging and Reports¶

The MacroServer allows to write into a log file log messages from the macros as well as additional log entries that enclose each macro execution, e.g. when it starts and when it ends.

This log feature is controlled via environment variables, allowing the activation/deactivation and the setting of the number of backup files, log directory and file names and output format.

Additional filters to this log messages can be applied with LOG_MACRO_FILTER.

The macro logmacro activate/deactivate the macro logging.

Report messages can be sent by the macros to a report file. This report feature is controlled by the MacroServer configuration.
