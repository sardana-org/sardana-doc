poolutil¶

Pool utils

Singletons

PoolUtil = <sardana.pool.poolutil._PoolUtil object>¶

Singleton instance of the _PoolUtil class.

It is a factory of tango.DeviceProxy objects and ensures only one instance of such objects is created for the whole process. Please refer to the _PoolUtil API on the available methods.
